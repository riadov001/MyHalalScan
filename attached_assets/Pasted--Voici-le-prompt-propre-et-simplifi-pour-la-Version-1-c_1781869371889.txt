Crée une application React Native Expo nommée HalalScan.
Objectif Version 1 : Application très simple pour personnes âgées ou malvoyantes. On scanne un code-barres → résultat immédiat avec couleurs plein écran + voix + sons (presque aucun texte).
UX/UI obligatoire :
	•	Fond très sombre.
	•	Écran principal : gros bouton jaune vif “SCANNER” au centre + caméra en arrière-plan.
	•	Après scan :
	◦	Écran entier VERT vif avec gros ✅ → Voix : “Ce produit est halal”
	◦	Écran entier ROUGE vif avec gros ❌ → Voix : “Attention, ce produit n’est pas halal”
	◦	Écran JAUNE → Voix : “Vérification nécessaire”
	•	Utilise expo-speech pour la voix en français.
	•	Ajoute sons et vibrations.
Fonctionnalités :
	•	Scan code-barres avec react-native-vision-camera.
	•	Appel à l’API OpenFoodFacts : https://world.openfoodfacts.org/api/v2/product/{barcode}.json
	•	Analyse avec cette liste exacte d’ingrédients (aucune autre) :

    "distéarate de glycérine", "triacétine", "e1518",

  ]
};
	•	Priorité aux labels OpenFoodFacts (“halal”, “sans porc”, etc.).
	•	Cache local (AsyncStorage) des produits scannés.
	•	Bouton “Marquer comme OK” pour ajouter à la liste blanche locale.
Après création :
	•	Installe toutes les dépendances nécessaires.
	•	Rends l’interface minimaliste avec très gros éléments et contraste fort.
	•	Génère un build Android preview avec EAS pour avoir un APK installable.
	•	Donne les commandes exactes pour le build.
Commence maintenant.
